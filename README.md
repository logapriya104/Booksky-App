# Booksky-App
A simple Book Management web app built with HTML, CSS, and JavaScript that allows users to add and delete books using a popup form.

## 🚀 Features

* Add new books using a popup form
* Display added books on the page
* Delete books easily
* Simple and clean user interface
* Responsive design

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

```
index.html
style.css
script.js
README.md
```

## 🎯 How It Works

1. Click the **Add Book** button.
2. A popup form appears.
3. Enter the **book name, author name, and description**.
4. Click **Add** to display the book.
5. Click **Delete** to remove the book.
